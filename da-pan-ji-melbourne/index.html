大盘鸡墨尔本哪里吃？来 Box Hill 试试山西李家厨房

如果你正在搜索「大盘鸡墨尔本」，大概率是想找一份够香、够下饭、适合朋友一起分享的中式硬菜。大盘鸡不是那种小份精致菜，而是一道非常适合聚餐的菜：鸡肉、土豆、酱汁、香料和主食搭配在一起，热热闹闹端上桌，越吃越有满足感。

位于 Box Hill 的 山西李家厨房，主打扎实、有烟火气的中式家常和西北风味菜品。除了很多客人熟悉的羊蝎子火锅、羊杂汤、烧烤和手工水饺，大盘鸡也很适合加入聚餐菜单。

大盘鸡为什么适合聚餐？

大盘鸡的特点就是「够大、够香、够分享」。它不是一个人随便吃两口的小菜，而是适合放在桌子中间，几个人一起夹着吃的主菜。

鸡肉吸收了酱汁和香料的味道，土豆软糯入味，汤汁可以搭配面食或主食一起吃。对于喜欢重口味、喜欢热菜、喜欢中式聚餐氛围的人来说，大盘鸡是一道很容易让人记住的菜。

第一次吃大盘鸡，应该期待什么味道？

大盘鸡通常是咸香、微辣、酱香和香料味结合在一起。它不像火锅那样需要一直涮，也不像烧烤那样是小串小串吃，而是更像一份完整的主菜。

如果你喜欢鸡肉、土豆、浓郁酱汁和适合拌面的菜，大盘鸡会很适合你。

在 Box Hill 吃大盘鸡的好处

Box Hill 是墨尔本华人美食非常集中的区域，适合朋友聚餐、家庭晚餐、下班后约饭，也适合周末专门来吃一顿中餐。

如果你住在 Box Hill、Doncaster、Blackburn、Burwood、Glen Waverley 或 Melbourne CBD 附近，来 Box Hill 吃大盘鸡都比较方便。

大盘鸡 + 羊蝎子火锅 + 羊肉串 + 手工水饺 + 凉菜。

山西李家厨房的大盘鸡适合喜欢中式热菜、喜欢鸡肉和土豆、喜欢浓郁酱汁、喜欢朋友一起分享美食的人。如果你正在找「墨尔本大盘鸡」「Box Hill 大盘鸡」「大盘鸡 Melbourne」，可以把山西李家厨房加入你的美食清单。
